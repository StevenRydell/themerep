## 🔧 Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/StevenRydell/themerep.git
   ```

2. **Open showcase file:**
   ```bash
   open Martex_Theme/martex/dist/MARTEX_HERO_SHOWCASE.html
   ```

3. **Extract components:**
   - Browse the showcase to find desired hero section
   - Copy the HTML section from the source
   - Include necessary CSS and JS dependencies

## 📱 Responsive Breakpoints

The theme uses Tailwind CSS responsive prefixes:
- `xsm:max-sm:` - Extra small devices
- `sm:max-md:` - Small devices  
- `md:max-lg:` - Medium devices
- `lg:max-xl:` - Large devices
- `xl:` - Extra large devices
